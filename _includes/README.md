Imagine you’re a cute bunny delivering a package from the post office to Granny, only for her to open it up and happily display to you a box filled to the brim with freshly severed and bloody bear claws. But you don’t have time to ask her anything, because the post master has a quota for you to hit… and you? You have to earn your keep.

## To Start
Download the 0.1.0 Final Build from Github [Releases](https://github.com/CandleSoup-Studios/Odd-Contents/releases/tag/capstone).

Platform: PC only. Upon download, double click on the *Odd Content_0.1.0.exe* executable to start the game.

## Core Mechanic
Helping deliver packages to townsfolk to uncover the mystery of the Shade.
